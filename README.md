# Belajar Java Array

Repository ini berisi kumpulan latihan dan tugas sekolah mengenai implementasi **Array** dan manipulasi data menggunakan bahasa pemrograman Java.

## 📝 Daftar Isi / Struktur File
* `Array.java` & `MainArray.java` : Latihan dasar array 1 dimensi.
* `Array2.java` & `MainArray2.java` : Latihan manipulasi dan pengolahan elemen array.
* `Array3.java` & `MainArray3.java` : Implementasi array multi-dimensi (matriks).

## 🚀 Cara Menjalankan Program
1. Pastikan Java Development Kit (JDK) sudah terinstal di komputer Anda.
2. Buka terminal/command prompt di folder ini.
3. Kompilasi file utama yang ingin dijalankan, contoh:
   ```bash
   javac MainArray.java
   java MainArray
